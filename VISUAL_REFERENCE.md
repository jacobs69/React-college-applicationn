## 🎯 Color Usage

### By Component
- **Headers**: Dark (#1a1a2e)
- **Primary Buttons**: Blue (#3b82f6)
- **Secondary Buttons**: Light Blue (#e0e7ff)
- **Success**: Green (#10b981)
- **Warning**: Amber (#f59e0b)
- **Danger**: Red (#ff3b30)
- **Backgrounds**: Light Gray (#f8f9fa)
- **Cards**: White (#fff)

### By Role
- **Student**: Blue (#3b82f6)
- **Teacher**: Purple (#8b5cf6)
- **Admin**: Multi-color

## 📐 Spacing Reference

```
Header Padding:     16px vertical, 20px horizontal
Card Padding:       12-14px
Button Padding:     16px vertical, 20px horizontal
Gap Between Items:  12px
Section Margin:     24px
```

## 🎬 Animation Reference

### Landing Page
- Logo: Fade in + Scale (0.8 → 1.0) over 800ms
- Dots: Static

### Transitions
- Page changes: Instant (state-based)
- Button press: Immediate visual feedback

## 📱 Responsive Breakpoints

- **Mobile**: < 480px (default)
- **Tablet**: 480px - 768px
- **Large**: > 768px

## ✨ Visual Hierarchy

```
Level 1 (Most Important)
├─ Page Headers (28px, Bold, Dark)
├─ Primary Buttons (Blue, Large)
└─ Key Stats (Large Numbers)

Level 2 (Important)
├─ Section Titles (16px, Bold)
├─ Card Titles (14px, Bold)
└─ Secondary Buttons (Small)

Level 3 (Supporting)
├─ Body Text (14px, Regular)
├─ Labels (13px, Semi-bold)
└─ Helper Text (12px, Gray)

Level 4 (Minimal)
├─ Small Text (11px)
└─ Timestamps (11px, Light Gray)
```
